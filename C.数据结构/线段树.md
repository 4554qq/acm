# 线段树 (Segment Tree)

## 基本概念

线段树是一种**二叉树**，用于维护**区间信息**（区间和、区间最值、区间GCD等）。

- 每个节点代表一个区间 `[l, r]`
- 根节点代表整个数组区间 `[1, n]`
- 叶子节点代表单个元素 `[i, i]`
- 左右孩子：左半区间 `[l, mid]`，右半区间 `[mid+1, r]`

**适用场景**：区间查询 + 区间修改，均可做到 **O(log n)**。

**空间**：一般开 `4 * n` 大小数组。

**核心思路**：把 `l, r` 存在节点结构体里，递归时直接用 `tr[p].l` / `tr[p].r`，少传两个参数。

## 模板代码（核心板子）

```cpp
#include<iostream>
using namespace std;
#define N 50005
#define lc p<<1          // 左孩子
#define rc p<<1|1        // 右孩子

int n, num[N];
struct node {
    int l, r;    // 节点管辖的区间
    int sum;     // 区间和
    int add;     // 懒惰标记（区间加）
} tr[4 * N];

// ============ 建树 ============
void build(int p, int l, int r) {
    tr[p] = {l, r, num[l]};   // 初始化节点
    if (l == r) return;
    int m = (l + r) >> 1;
    build(lc, l, m);
    build(rc, m + 1, r);
    tr[p].sum = tr[lc].sum + tr[rc].sum;
}

// ============ 上传 ============
void pushup(int p) {
    tr[p].sum = tr[lc].sum + tr[rc].sum;
}

// ============ 下传懒惰标记 ============
void pushdown(int p) {
    if (tr[p].add) {
        tr[lc].sum += tr[p].add * (tr[lc].r - tr[lc].l + 1);
        tr[rc].sum += tr[p].add * (tr[rc].r - tr[rc].l + 1);
        tr[lc].add += tr[p].add;
        tr[rc].add += tr[p].add;
        tr[p].add = 0;
    }
}

// ============ 区间修改：[x, y] 每个数加 k ============
void update1(int p, int x, int y, int k) {
    if (x <= tr[p].l && tr[p].r <= y) {     // 完全覆盖
        tr[p].sum += (tr[p].r - tr[p].l + 1) * k;
        tr[p].add += k;
        return;
    }
    int m = (tr[p].l + tr[p].r) >> 1;
    pushdown(p);
    if (x <= m) update1(lc, x, y, k);
    if (y > m)  update1(rc, x, y, k);
    pushup(p);
}

// ============ 单点修改：位置 x 加 k ============
void update(int p, int x, int k) {
    pushdown(p);
    if (tr[p].l == x && tr[p].r == x) {
        tr[p].sum += k;
        return;
    }
    int m = (tr[p].l + tr[p].r) >> 1;
    if (x <= m) update(lc, x, k);
    else        update(rc, x, k);
    tr[p].sum = tr[lc].sum + tr[rc].sum;
}

// ============ 区间查询：[x, y] 的和 ============
int quary(int p, int x, int y) {
    if (x <= tr[p].l && tr[p].r <= y)   // 完全覆盖
        return tr[p].sum;
    int m = (tr[p].l + tr[p].r) >> 1;
    int sum = 0;
    if (x <= m) sum += quary(lc, x, y);
    if (y > m)  sum += quary(rc, x, y);
    return sum;
}

// ============ 主函数（测试建树） ============
int main() {
    cin >> n;
    for (int i = 1; i <= n; i++) cin >> num[i];
    build(1, 1, n);
    for (int i = 1; i <= 4 * n; i++) cout << tr[i].sum << endl;
    return 0;
}
```

## 代码详解

### 1. 结构体设计

```cpp
struct node {
    int l, r;    // 管辖区间 [l, r]（存在节点里，递归不用传参）
    int sum;     // 区间和
    int add;     // 懒惰标记
} tr[4 * N];
```

把 `l, r` 存在节点里的好处：`update` / `update1` / `quary` 不需要再传 `l, r` 参数，直接用 `tr[p].l` / `tr[p].r`。

### 2. 建树 `build`

`tr[p] = {l, r, num[l]}` 一步初始化三个字段。叶子节点 `l == r` 时直接返回；非叶子递归建左右子树后合并 `sum`。

### 3. 上传 `pushup`

```cpp
void pushup(int p) {
    tr[p].sum = tr[lc].sum + tr[rc].sum;
}
```

子节点更新后，重新计算父节点值。**区间修改后必须调 `pushup(p)`**。

### 4. 下传 `pushdown`（懒惰标记核心）

```cpp
void pushdown(int p) {
    if (tr[p].add) {
        // 子节点 sum += 标记值 × 子区间长度
        tr[lc].sum += tr[p].add * (tr[lc].r - tr[lc].l + 1);
        tr[rc].sum += tr[p].add * (tr[rc].r - tr[rc].l + 1);
        // 标记下传
        tr[lc].add += tr[p].add;
        tr[rc].add += tr[p].add;
        // 清空当前标记
        tr[p].add = 0;
    }
}
```

"先记账，用到时再下传"——进入子节点前调用 `pushdown`，保证子树数据正确。

### 5. 区间修改 `update1`

三步走：
1. **完全覆盖** → 直接更新 `sum`，打上 `add` 标记，return
2. **部分覆盖** → 先 `pushdown`，再递归左右子区间
3. 回溯时 **`pushup`** 更新父节点

### 6. 单点修改 `update`

即使是单点修改，也要先 `pushdown`（因为可能有未下传的标记影响当前节点），然后找到叶子更新。

### 7. 区间查询 `quary`

递归思路：当前区间被完全覆盖直接返回；否则分左右子区间查，累加结果。

## 操作拉通流程示例

以数组 `[1, 5, 3, 7, 2]` 为例：

```
                [1,5] sum=18
              /           \
        [1,3] sum=9      [4,5] sum=9
        /       \         /       \
    [1,2] s=6 [3,3] s=3 [4,4] s=7 [5,5] s=2
    /     \
 [1,1]   [2,2]
  s=1     s=5
```

- **区间查询 [2,4]** → 访问 [1,5]→右半覆盖[1,3]→左半部分[1,2]、[3,3] 完全覆盖、[4,4] 完全覆盖 → sum = 5+3+7 = 15
- **区间修改 [2,3] +2** → [1,3] 完全覆盖，sum+4, add+2，[1,5] pushup → sum=22

## 常见变体

| 维护内容    | pushup 合并方式                        | add 标记含义   |
|------------|---------------------------------------|---------------|
| 区间和      | `tr[p].sum = tr[lc].sum + tr[rc].sum` | 区间加的值     |
| 区间最大值   | `tr[p].mx = max(tr[lc].mx, tr[rc].mx)` | 区间加的值     |
| 区间最小值   | `tr[p].mn = min(tr[lc].mn, tr[rc].mn)` | 区间加的值     |

换成最值只需改 `pushup` 和 `quary` 的合并逻辑，框架不变。

## 例题

### 例题1：P3372 【模板】线段树 1（区间加 + 区间求和）

[P3372 【模板】线段树 1](https://www.luogu.com.cn/problem/P3372)

**关键修改**：输入输出适配，`num` → `a`，`int` → `long long`。

```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;
const int N = 100005;
#define lc p<<1
#define rc p<<1|1

ll a[N];
struct node {
    int l, r; ll sum, add;
} tr[4 * N];

void build(int p, int l, int r) {
    tr[p] = {l, r, a[l], 0};
    if (l == r) return;
    int m = (l + r) >> 1;
    build(lc, l, m);
    build(rc, m + 1, r);
    tr[p].sum = tr[lc].sum + tr[rc].sum;
}

void pushup(int p) {
    tr[p].sum = tr[lc].sum + tr[rc].sum;
}

void pushdown(int p) {
    if (tr[p].add) {
        tr[lc].sum += tr[p].add * (tr[lc].r - tr[lc].l + 1);
        tr[rc].sum += tr[p].add * (tr[rc].r - tr[rc].l + 1);
        tr[lc].add += tr[p].add;
        tr[rc].add += tr[p].add;
        tr[p].add = 0;
    }
}

void update(int p, int x, int y, ll k) {
    if (x <= tr[p].l && tr[p].r <= y) {
        tr[p].sum += k * (tr[p].r - tr[p].l + 1);
        tr[p].add += k;
        return;
    }
    pushdown(p);
    int m = (tr[p].l + tr[p].r) >> 1;
    if (x <= m) update(lc, x, y, k);
    if (y > m)  update(rc, x, y, k);
    pushup(p);
}

ll query(int p, int x, int y) {
    if (x <= tr[p].l && tr[p].r <= y) return tr[p].sum;
    int m = (tr[p].l + tr[p].r) >> 1;
    ll res = 0;
    if (x <= m) res += query(lc, x, y);
    if (y > m)  res += query(rc, x, y);
    return res;
}

int main() {
    int n, m;
    cin >> n >> m;
    for (int i = 1; i <= n; i++) cin >> a[i];
    build(1, 1, n);
    while (m--) {
        int op, x, y; ll k;
        cin >> op >> x >> y;
        if (op == 1) {
            cin >> k;
            update(1, x, y, k);
        } else {
            cout << query(1, x, y) << '\n';
        }
    }
    return 0;
}
```

### 例题2：P3373 【模板】线段树 2（区间乘 + 区间加 + 区间求和）

[P3373 【模板】线段树 2](https://www.luogu.com.cn/problem/P3373)

**关键**：两个标记 `mul` 和 `add`，**先乘后加**。

```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;
const int N = 100005;
#define lc p<<1
#define rc p<<1|1

ll a[N], mod;
struct node {
    int l, r;
    ll sum, mul, add;
} tr[4 * N];

void build(int p, int l, int r) {
    tr[p] = {l, r, a[l] % mod, 1, 0};
    if (l == r) return;
    int m = (l + r) >> 1;
    build(lc, l, m);
    build(rc, m + 1, r);
    tr[p].sum = (tr[lc].sum + tr[rc].sum) % mod;
}

void pushup(int p) {
    tr[p].sum = (tr[lc].sum + tr[rc].sum) % mod;
}

void pushdown(int p) {
    // 左孩子：先乘后加
    tr[lc].sum = (tr[lc].sum * tr[p].mul + tr[p].add * (tr[lc].r - tr[lc].l + 1)) % mod;
    tr[lc].mul = tr[lc].mul * tr[p].mul % mod;
    tr[lc].add = (tr[lc].add * tr[p].mul + tr[p].add) % mod;
    // 右孩子
    tr[rc].sum = (tr[rc].sum * tr[p].mul + tr[p].add * (tr[rc].r - tr[rc].l + 1)) % mod;
    tr[rc].mul = tr[rc].mul * tr[p].mul % mod;
    tr[rc].add = (tr[rc].add * tr[p].mul + tr[p].add) % mod;
    // 清空
    tr[p].mul = 1; tr[p].add = 0;
}

void update_mul(int p, int x, int y, ll k) {
    if (x <= tr[p].l && tr[p].r <= y) {
        tr[p].sum = tr[p].sum * k % mod;
        tr[p].mul = tr[p].mul * k % mod;
        tr[p].add = tr[p].add * k % mod;
        return;
    }
    pushdown(p);
    int m = (tr[p].l + tr[p].r) >> 1;
    if (x <= m) update_mul(lc, x, y, k);
    if (y > m)  update_mul(rc, x, y, k);
    pushup(p);
}

void update_add(int p, int x, int y, ll k) {
    if (x <= tr[p].l && tr[p].r <= y) {
        tr[p].sum = (tr[p].sum + k * (tr[p].r - tr[p].l + 1)) % mod;
        tr[p].add = (tr[p].add + k) % mod;
        return;
    }
    pushdown(p);
    int m = (tr[p].l + tr[p].r) >> 1;
    if (x <= m) update_add(lc, x, y, k);
    if (y > m)  update_add(rc, x, y, k);
    pushup(p);
}

ll query(int p, int x, int y) {
    if (x <= tr[p].l && tr[p].r <= y) return tr[p].sum;
    pushdown(p);   // 查询前下传标记
    int m = (tr[p].l + tr[p].r) >> 1;
    ll res = 0;
    if (x <= m) res = (res + query(lc, x, y)) % mod;
    if (y > m)  res = (res + query(rc, x, y)) % mod;
    return res;
}
```

### 例题3：P1531 I Hate It（单点修改 + 区间最大值）

[P1531 I Hate It](https://www.luogu.com.cn/problem/P1531)

**关键**：`sum` → `mx`，合并改 `max`，单点修改无需 lazy。

```cpp
#include <bits/stdc++.h>
using namespace std;
const int N = 200005;
#define lc p<<1
#define rc p<<1|1

int a[N];
struct node {
    int l, r, mx;
} tr[4 * N];

void build(int p, int l, int r) {
    tr[p] = {l, r, a[l]};
    if (l == r) return;
    int m = (l + r) >> 1;
    build(lc, l, m);
    build(rc, m + 1, r);
    tr[p].mx = max(tr[lc].mx, tr[rc].mx);
}

void pushup(int p) {
    tr[p].mx = max(tr[lc].mx, tr[rc].mx);
}

void update(int p, int pos, int val) {
    if (tr[p].l == pos && tr[p].r == pos) {
        tr[p].mx = max(tr[p].mx, val);
        return;
    }
    int m = (tr[p].l + tr[p].r) >> 1;
    if (pos <= m) update(lc, pos, val);
    else          update(rc, pos, val);
    pushup(p);
}

int query(int p, int x, int y) {
    if (x <= tr[p].l && tr[p].r <= y) return tr[p].mx;
    int m = (tr[p].l + tr[p].r) >> 1;
    int res = 0;
    if (x <= m) res = max(res, query(lc, x, y));
    if (y > m)  res = max(res, query(rc, x, y));
    return res;
}
```

## 复杂度

| 操作       | 时间复杂度 |
|------------|-----------|
| 建树       | O(n)      |
| 单点修改   | O(log n)  |
| 区间修改   | O(log n)  |
| 区间查询   | O(log n)  |
| 空间       | O(4n)     |

## 注意事项

1. **数组开 4 倍**：`tr[4 * N]`，不是 2 倍。
2. **下标从 1 开始**：方便 `lc = p<<1`、`rc = p<<1|1` 找孩子。
3. **开 long long**：区间求和容易爆 int。
4. **区间修改后记得 `pushup(p)`**：回溯时更新父节点，不然祖先数据是旧的。
5. **进入子节点前 `pushdown(p)`**：不下传的话子树数据是旧的。单点修改也要先 pushdown（可能有未下传的标记）。
6. **查询时是否需要 pushdown？** 区间求和可以不调（因为 add 只影响子孙，不影响当前节点的 sum）；但带乘法标记时必须调。
